# STAVRA

Спортивный журнал тренировок с профилями, историей пробежек, маршрутами по Москве и публичной страницей отчета.

## Запуск

1. Установить зависимости:

```bash
npm install
```

2. Создать `.env` по примеру `.env.example`.

3. Поднять PostgreSQL:

```bash
docker compose up -d
```

4. Создать таблицы и Prisma Client:

```bash
npm run prisma:migrate
```

5. Запустить сайт:

```bash
npm run dev
```

По умолчанию приложение открывается на `http://localhost:3000`.

## Что уже есть

- регистрация и вход;
- гостевой режим без регистрации;
- локальная история в браузере без записи в БД;
- перенос локальной истории в профиль после входа;
- отдельная история для каждого пользователя;
- создание пробежки за выбранную дату и время;
- дефолтная длительность занятия 90 минут;
- районы Москвы и fallback-маршруты;
- ручная постановка точек на карте-превью;
- расчет дистанции, темпа и калорий;
- публичная страница тренировки;
- явная пометка типа записи: `Manual`, `Planned`, `Recorded`.
